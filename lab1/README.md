## Секции
- Header: логотип и меню‑якоря (`#about`, `#skills`, `#portfolio`, `#contacts`).
- Hero: приветствие и кнопки.
- About: текст и список фактов.
- Skills: карточки с «прогрессом» навыков.
- Portfolio: карточки с изображениями и описанием.
- Contacts: контакты списком.
- Footer: копирайт и ссылки.

## Тема и стили
В `css/styles.css` используются CSS‑переменные (минималистичная розовая тема):
css
:root {
  --bg: #ffffff; --text: #262126; --muted: #6b5a6b;
  --primary: #ff4da6; --primary-600: #e64495;
  --card: #ffffff; --border: #ffe0ee;
}

## Адаптивность
- Брейкпоинты: 768px (tablet) и 1024px (desktop).
- На мобильных — одна колонка; на планшетах/десктопах — сетки в 2–4 колонки.
